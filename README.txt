Untuk menjalankan file program ini, berikut adalah petunjuk pakainya:
1. Install nodejs
2. buka folder project ini menggunakan cmd / terminal
3. lalu ketikkan perintah npm install (kalo error hapus dulu folder node_modules nya)
4. lalu jalankan perintah berikut pada cmd "npm run server" (tanpa tanda petik)
5. terakhir akses situs localhost:8000 menggunakan browser kesayangan anda
